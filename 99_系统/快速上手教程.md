# 快速上手教程

> 跟着走一遍，体验文渊的核心工作流：**想法 → 计划 → 项目 → 执行 → 积累**。大约 20 分钟。

---

## 准备

1. 在 Obsidian 中打开文渊文件夹
2. 打开 Terminal 插件的终端面板（见 [[Obsidian新手指南]]）
3. 在终端中运行 `claude`

---

## Step 1：开始新的一天

每天的第一件事：

```
/start-my-day
```

AI 会：
- 读取你昨天的日记和未完成事项
- 问你今天的研究重点和新想法
- **推送学术动态**（根据你的偏好设置，自动搜索领域内新论文、教职、会议）
- 生成今天的日记，自动把新想法存入收件箱

第一次使用时还没有历史数据，AI 会帮你创建第一篇日记。

---

## Step 2：捕捉一个想法

假设你在课上想到一个有趣的问题："中国当代文学在海外传播中是否存在'东方主义'式的刻板印象？"

你可以：
- 直接在 `00_收件箱/` 中新建文件记录
- 或在 `/start-my-day` 的对话中说出来，AI 自动创建收件箱条目

收件箱是**灵感快速捕捉站**——想到什么就扔进去，不需要整理。

---

## Step 3：头脑风暴

这个想法还比较模糊，用头脑风暴梳理一下：

```
/brainstorm
```

AI 会用开放性问题引导你思考：
- "你对这个问题的哪个方面最感兴趣？"
- "想从什么角度切入？"
- "已有研究做过类似探索吗？"

聊完后，AI 整理出讨论要点和可能的研究方向。

---

## Step 4：探索核心概念

头脑风暴中提到了"东方主义"和"文化霸权"，深入了解一下：

```
/explore 文化霸权
```

AI 生成一张概念卡片 → 保存到 `40_概念/文化霸权.md`，包含定义、学术脉络、代表人物、相关概念的双向链接。

---

## Step 5：系统调研

想对这个选题做一次完整的文献调研：

```
/research 中国文学海外传播与东方主义
```

AI 先生成调研计划让你确认，然后系统性执行：
- 生成研究综述 → `30_文献/`
- 创建/更新概念卡片 → `40_概念/`
- 推荐关键文献

---

## Step 6：精读一篇关键文献

调研中发现赛义德的《东方主义》是绕不开的文献：

```
/read
```

AI 用苏格拉底式提问引导你精读："核心论点是什么？""方法论是什么？""对你的研究有什么启发？"

对话后生成文献笔记 `赛义德_1978_东方主义.md` → `30_文献/`

---

## Step 7：把想法变成项目

经过调研和精读，想法已经足够清晰，正式立项：

```
/kickoff
```

AI 引导你填写研究问题、意义、阶段计划，生成项目文件 → `20_项目/`。

---

## Step 8：快速问一个问题

写项目计划时，想快速了解"再现理论"：

```
/ask 什么是再现理论？
```

AI 直接给简洁回答，不生成大量文档。这就是 `/ask` 的用法——写作中随手查概念，轻量快速。

---

## Step 9：写作辅助

写了一段研究背景，想让 AI 优化：

```
/write
```

AI 改善学术表达、检查逻辑、建议用词——保留你的核心观点。

---

## Step 10：结束一天

工作结束时：

```
/end-my-day
```

AI 会：
- 汇总你今天的成果（新建了哪些笔记、推进了哪些项目）
- 更新项目的 Progress 记录
- 归档今日对话摘要
- 生成明日重点

---

## Step 11：定制你自己的工作流

用了一段时间后，你发现经常需要把课堂纸质笔记 OCR 归档。这是一个重复性任务——可以把它做成你自己的 skill！

在 `.claude/skills/` 下新建 `ocr-notes/SKILL.md`：

```markdown
---
name: ocr-notes
description: 课堂笔记 OCR — 拍照识别手写笔记并归档到 vault
---

# 课堂笔记 OCR 归档

## 步骤
1. 用户提供笔记照片或文字
2. 识别/整理内容
3. 分析内容类型（文献信息、概念、研究想法）
4. 按类型创建对应笔记到正确位置
5. 建立 wikilinks 关联
```

保存后输入 `/ocr-notes` 就可以触发了。

> 这就是文渊的**个性化定制**：你的使用习惯可以沉淀为可复用的工作流。更多定制方法见 [[文渊使用指南#个性化定制]]。

---

## 回顾

你体验了文渊的完整闭环：

```
想法  ──→  /brainstorm  ──→  清晰方向
  ↓
计划  ──→  /explore + /research  ──→  理论基础
  ↓
立项  ──→  /kickoff  ──→  结构化项目
  ↓
执行  ──→  /read + /write  ──→  文献 + 文本
  ↓
积累  ──→  /end-my-day  ──→  记录 + 归档
  ↓
定制  ──→  自建 skills  ──→  个人工作流
```

随着使用，你的文渊会积累越来越多的文献笔记、概念卡片和项目记录，通过双向链接编织成属于你的学术知识网络。

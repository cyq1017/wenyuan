# 快速上手教程：纸质笔记变学术知识

> 通过一个真实场景——把课堂上的纸质笔记电子化并归档到文渊，带你认识文渊的核心功能。大约需要 20 分钟。

---

## 场景设定

你刚上完一节关于"后殖民文学理论"的课。笔记本上记了满满两页：老师提到了赛义德的《东方主义》、斯皮瓦克的底层研究、还有一个你没太听懂的概念"文化霸权"。

现在你想把这些纸质笔记整理到文渊里，变成可以检索、关联、积累的学术知识。

---

## 准备工作

1. 确保已安装 Obsidian 和 Claude Code（见 [[Obsidian新手指南]] 和 [[Claude Code新手指南]]）
2. 在 Obsidian 中打开文渊
3. 在 Obsidian 中打开 Terminal 插件的终端面板
4. 在终端中运行 `claude`

---

## Step 1：拍照 → OCR → 文字

先把纸质笔记变成电子文字。

**方法A：用手机自带 OCR（最简单）**
1. iPhone：打开相机拍照 → 长按图片中的文字 → "拷贝全部"
2. Android：用 Google Lens 扫描 → 复制文字

**方法B：用 AI 识别**
1. 把笔记拍照，放到文渊的 `00_收件箱/` 文件夹里
2. 在 Claude Code 中说："帮我识别这张图片上的手写笔记内容"
3. Claude 会使用视觉能力识别文字并输出

假设你得到了这样的文字：

```
后殖民文学理论 2026-04-06

- 赛义德《东方主义》1978：西方对东方的再现是一种权力话语
- 斯皮瓦克：底层能说话吗？关注被压制的声音
- 文化霸权（葛兰西）→ 后殖民研究中的应用
- 课后思考：中国文学在全球传播中是否也受到"东方主义"式的定型？
```

---

## Step 2：扔进收件箱

在 Obsidian 中，在 `00_收件箱/` 文件夹下新建一个文件 `后殖民课堂笔记.md`，粘贴刚才的文字。

或者更简单——直接在 Claude Code 中说：

> "帮我在收件箱里创建一条笔记，内容是：[粘贴你的文字]"

---

## Step 3：让 AI 整理笔记

在 Claude Code 中输入：

```
/organize
```

然后指向你刚才创建的收件箱笔记。AI 会分析其中的内容，并提出整理方案：

> "我在这段笔记中发现了以下内容：
> 1. 关于赛义德《东方主义》的文献信息 → 建议创建文献精读笔记
> 2. '文化霸权'这个概念 → 建议创建概念卡片
> 3. 关于中国文学与东方主义的研究灵感 → 建议保留在收件箱
>
> 你觉得这样处理可以吗？"

确认后，AI 会自动创建对应的笔记。

---

## Step 4：深入探索一个概念

课堂上"文化霸权"没听太懂。在 Claude Code 中输入：

```
/explore 文化霸权
```

AI 会为你生成一张详细的概念卡片，包含：
- **定义**：葛兰西提出的概念，指支配阶级通过文化手段建立共识
- **学术脉络**：从葛兰西到后殖民研究的应用
- **代表人物**：葛兰西、威廉斯、霍尔等
- **相关概念**：`[[意识形态]]`、`[[文化资本]]`、`[[后殖民主义]]`

卡片保存到 `40_概念/文化霸权.md`，并自动和你的课堂笔记建立链接。

---

## Step 5：精读一篇关键文献

赛义德的《东方主义》是核心参考文献。在 Claude Code 中输入：

```
/read
```

告诉 AI 你想精读赛义德的《东方主义》。AI 会用提问引导你思考：

- "你觉得赛义德最核心的论点是什么？"
- "他用什么方法论来支撑？"
- "这本书和你的课堂主题有什么关系？"

对话结束后，AI 生成一份结构化的文献笔记 `赛义德_1978_东方主义.md`，保存到 `30_文献/`。

---

## Step 6：把灵感变成项目

课堂上想到的问题——"中国文学在全球传播中是否受到东方主义式定型"——可能是一个值得研究的选题。

```
/kickoff
```

AI 引导你明确研究问题、评估可行性，最终生成一个项目文件保存到 `20_项目/`。

---

## Step 7：快速问一个问题

写项目计划时，你想快速了解"再现理论"是什么：

```
/ask 什么是再现理论？它和后殖民研究有什么关系？
```

AI 直接给你简洁的回答（不生成大量文档），并问你要不要进一步 `/explore` 这个概念。

这就是 `/ask` 的用法——**轻量级学术问答**，适合写作中随手查一个概念，不需要完整的概念卡片。

---

## Step 8：润色一段写作

你在项目计划中写了一段研究背景，想让 AI 帮你优化：

```
/write
```

粘贴你的草稿，AI 会：
- 改善学术表达
- 检查论证逻辑
- 建议更好的用词
- 保留你的核心观点

---

## Step 9：结束一天的工作

回到今天的日记（`10_日记/` 下），在"当日总结"部分记录：

```

- 初步立项：中国文学与东方主义

未完成:

明日重点:
1. 继续精读赛义德
2. 搜索相关文献
```

## 回顾：你完成了什么？

从一份纸质课堂笔记出发，你的知识库中新增了：

```
00_收件箱/后殖民课堂笔记.md     ← 已处理
30_文献/赛义德_1978_东方主义.md  ← 精读笔记
40_概念/文化霸权.md              ← 概念卡片
20_项目/中国文学与东方主义.md    ← 新项目

```

这些笔记通过 `[[双向链接]]` 互相关联。随着你的积累，文渊会变成一个越来越丰富的个人学术维基。

# 欢迎使用文渊！

> 文渊 = Obsidian（知识库） + AI 助手（Claude Code 等）。你现在已经在 Obsidian 中打开了文渊，接下来只需要几步就可以开始使用。

---

## 第一步：确认 AI 助手已安装

文渊需要一个 AI 编码助手来驱动工作流。如果你还没有安装，先完成安装：

| AI 工具 | 安装指南 |
|---------|---------|
| **Claude Code**（推荐） | 查看 [[Claude Code新手指南]] |
| 其他工具（Kimi Code / Codex / Gemini CLI） | 查看 [[README]] 中的"其他 AI 工具"部分 |

> 已经装好了？跳到第二步 👇

## 第二步：安装 Terminal 插件

在 Obsidian 里打开终端，这样你可以边看笔记边和 AI 对话：

`设置` → `第三方插件` → `浏览` → 搜索 **Terminal** → 安装并启用

> 详细步骤和其他推荐插件见 [[Obsidian新手指南]]

## 第三步：开始你的第一天

1. 打开终端：`Cmd/Ctrl + P` → 搜索 `Terminal: Open terminal`
2. 在终端中输入 `claude`（或你安装的其他 AI 工具）
3. 输入 `/start-my-day`

AI 会引导你完成偏好设置（首次使用时），然后开始你的第一天！

# 欢迎使用文渊！

> 第一次使用？按顺序完成以下步骤：

1. ✅ 阅读本文件（你已经在这里了）
2. 📖 打开 [[快速上手教程]] — 20 分钟体验完整工作流
3. ⚙️ 编辑 [[偏好设置]] — 设定你的研究领域（也可以在首次 `/start-my-day` 时由 AI 引导设置）
4. 📚 需要帮助？查看 [[文渊使用指南]]

---

## 快速开始

在终端中运行：

```
claude
```

然后输入：

```
/start-my-day
```

AI 会引导你完成偏好设置（首次使用时），然后开始你的第一天！
